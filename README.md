# note_book

## Notebook App
Notebook App is our First Project That Practical Provider State Management Using Flutter Framework.
Notebook Image Logo is Generated By & Using Microsoft Bing Image AI.
Notebook is Now Connected to Sqflite Mobile And Sqflite Common ffi Desktop
You Can Make CRUD Operations         You Can Create Update or Delete Notes with or Without Image(add Default Notebook Logo Image if there no image) And Read it.Features:
1-Customize Note Content Like Changing Color or Font Size or Font Weight.
2-Search Note using noteTitle or noteContent.
3-Showing Time in Real Time.
4-Write Long Text in noteContent!5-Support Android Platform And Windows Platform!.
6-Chossing Categories Notes Like Study Personal Others Works.
7-Featured Feature That Only Show Notes in Study Categories.
8-Select Category For Note Ex:Personal,Study,Work,Others Category With Default Colors You Can Also Select Category And Changing The Default Color!.
9-Automatically it Fetch Current Date & Time when you create note.
10-Support Copy & Paste in NotesYou can Select your Notes That you Want to Copy & Paste it in Anywhere.
11-Because It Using Sqflite & Sqflite Common ffi This App Works Offline!.
Packages Used in This Project:1-Sqflite That Support SQLite For Android & IOS & MacOS.
2-Sqflite Common ffi That Support SQLite using DatabaseFactoryFfi in Desktop.
3-Path Provider
4-Path
5-awesome_dialog
6-Image Picker
7-Provider State Management 
8-Clock ⏰.
